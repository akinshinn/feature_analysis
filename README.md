Полезность входных переменных

Описание проекта

Проект посвящён исследованию зависимости между выходной переменной y и набором входных признаков x₁, x₂, …, x₁₀.
Данные предоставлены в виде таблицы (task2.txt), где:

первые 10 столбцов — входные переменные (факторы);

11-й столбец — выходная переменная y.

Цель проекта — определить, какие из входных признаков наиболее информативны при построении модели для прогноза $y$.
